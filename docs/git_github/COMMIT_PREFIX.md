# コミットプレフィックス
コミットメッセージには以下のいずれかのプレフィックスを付ける

- feat: 新機能の実装
- fix: バグ修正
- docs: ドキュメントの作成・編集
- format: コードフォーマットの修正 (white-space, formatting, missing semi-colons, etc)
- refactor: 機能実装やバグ修正を伴わない変更（リファクタリング）
- perf: パフォーマンスを向上させる変更
- test: テストの実装・編集
- chore: ビルドプロセスの変更，npm scriptの追加・変更，各種設定など
